# Frontend

Using [Ant Design Pro](https://pro.ant.design/docs/getting-started).


## Environment Prepare

Install `node_modules`:

```bash
tyarn
```

### Start project

```bash
tyarn start
```

### Build project

```bash
tyarn build
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### Test code

```bash
npm test
```
